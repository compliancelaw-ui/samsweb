SAM'S OATH MAP PLUGIN
====================

This plugin creates an interactive map showing OATH-takers and integrates with Gravity Forms.

INSTALLATION:
1. Upload this folder to WordPress
2. Activate the plugin
3. Go to Settings → Sam's OATH Map
4. Add your Mapbox access token (get free at mapbox.com)

REQUIREMENTS:
- Gravity Forms plugin (for form submissions)
- Mapbox account (free tier works fine)

USAGE:
1. Create Gravity Forms for OATH and Story submissions
2. Add map to any page using shortcode: [samsoath_map]
3. Pins automatically appear when forms are submitted

FEATURES:
- Automatic geocoding of city/state to coordinates
- Three pin colors for different categories
- Filterable map by category
- Story linking when users share stories
- Privacy-first (only city/state shown publicly)
